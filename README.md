## Prerequisites

- Node (of course)
- Yeoman : `npm i -g yeoman`

## Installation

`npm i -g @dwmaj/generator-bootstrap`

## All-in-one (via `npx`, it can take several minutes…)

`npx -p yo -p @dwmaj/generator-bootstrap -c 'yo @dwmaj/bootstrap'`

## Usage

```bash
cd ~/projects
yo @dwmaj/bootstrap
```
